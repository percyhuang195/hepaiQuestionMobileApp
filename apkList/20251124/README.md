# HePai 題庫整合系統 20251124
## 本次更新要點
### 更新內容：
- 增加題庫的題目數量至18題
- 作答時可以顯示圖片(考卷上的副圖、譜例)
- 開發出主頁面(僅文字UI，不具備統計功能)
### 修正的問題：
- 解決題目詳解的選項編號(1、2、3、4)跑版問題
- 解決選項文字過多時，畫面會顯示警告線(爆版提示)的問題
## 主要資訊：
### 功能：
- 讀取JSON檔案內的題目資料(目前設定18個問題，題目資料包含問題、選項、圖片)，隨機顯示在畫面上
- 當使用者點擊任意選項後，跳出正確答案，並可以讓使用者決定要「查看詳解」或者「繼續作答」
### 限制：
- 題目無法直接透過APP內功能進行調整
- 版面和功能都非常陽春
- 主頁面只有文字UI，沒有統計功能
### 未來發展：
- 增加更多題目到JSON檔案內，或整合Web API進行題目資料串接
- 讓使用者可以直接透過APP新增、修改、刪除題目(可以連線至資料庫)
- 增加統計系統，讓使用者可以針對錯題進行複習
- 可以選擇作答模式，讓使用者決定是否在作答完後立刻顯示答案
- 美化APP版面
- 在詳解中增加更多可以強化記憶的媒介(例如音檔播放、更多圖片)
- 增加一個圖表頁面，可以顯示今天刷題的題數
- 可以統計每天刷題的數量，和已經作答過、尚未作答的題目
