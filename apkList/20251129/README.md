# HePai 題庫整合系統 20251129
## 本次更新要點
### 修正的問題：
- 修正圖表介面重複顯示資料的問題
- 修正題庫在尚未回答任何問題時，清單全數不顯示的問題
- 修正題庫頁面清單沒有置中的問題
- 透過鎖定螢幕方向，解決螢幕轉向時的爆版問題
## 主要資訊：
### 功能：
- 開啟APP時，主頁面顯示摘要(作答紀錄、題庫檢視進度)
- 透過點按首頁中的「題庫解鎖進度」區塊開啟題庫瀏覽介面，讓使用者可以逐題檢視題庫
- 透過點按首頁中的「今日作答題數、總計作答題數」區塊開啟圖表介面，讓使用者可以檢視作答紀錄
- 讀取JSON檔案內的題目資料(目前設定18個問題，題目資料包含問題、選項、圖片)，隨機顯示在畫面上
- 當使用者點擊任意選項後，跳出正確答案，並可以讓使用者決定要「查看詳解」或者「繼續作答」
### 未來發展：
- 增加更多題目到JSON檔案內，或整合Web API進行題目資料串接
- 讓使用者可以直接透過APP新增、修改、刪除題目(可以連線至資料庫)
- 美化APP版面
- 在詳解中增加更多可以強化記憶的媒介(例如音檔播放、更多圖片)
- 可以選擇作答模式，讓使用者決定是否在作答完後立刻顯示答案
