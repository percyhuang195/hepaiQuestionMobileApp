# HePai 題庫整合系統
- 一個為「大學音樂術科考試」中的選擇題庫，設計的問答/題庫系統，目前支援Android 8以上的安卓裝置。
## 可運行範例
- Android APK：請到apkList資料夾中，找到最新版本(或者任意一個你想玩的版本)，並將APK檔案安裝到手機內。
- 版本限制：理論上Android 8.0以上的大多數手機都可以安裝，若在安裝/使用上遇到問題，請發Issue給我們。
## APK檔案安裝說明
### 透過USB連線電腦進行安裝
1. 先下載APK檔案到電腦上
1. 使用USB連接線將Android裝置和電腦連線
1. 透過電腦將APK檔案移動到手機的「下載」資料夾內
1. 到Android裝置上，找到「Files」程式(或者在手機上具備檔案總管功能的APP)
1. 找到「下載」資料夾，並點選APK檔案
1. 依照指示進行下載權限設定(通常是要打開APK下載授權，讓手機可以從Google Play以外的地方下載程式)
1. 如果顯示「安裝完成」這類的的字樣，恭喜你成功的安裝了這個APP，如果在安裝過程中發生問題，請發Issue給我們。
